# advent of code 2021

## Before you start

Gotta install dependencies:

```sh
$ npm install
```

## How to do the thing

```sh
$ node ./index.js [DAY NUMBER]
# e.g.: node ./index.js 1
```

Or you could also:

```sh
$ npm start -- [DAY NUMBER]
# e.g.: npm start -- 1
```

You got options, is all I'm sayin'.
